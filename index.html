Jakie mamy rozmiary w CSS?

  1. px - piksele
  2. in - cale : 1in == 96px
  3. cm - centymetry : 1cm == 37.8px
  4. mm - milimetry 1mm == 0.1cm == 3.78px

Rozmiary dla czionek : em

  1. em - bazuje na wielkości litery M
  2. 1em == 16px == 0.17in == 12pt == 1pc == 4.2mm == 0.42cm

Jakie mamy rozmiary w CSS?

  1. %

vw,vh : viewport

  1. 1 vw jest równy 1% długości ekranu
  2. 1 vh jest równy 1% wysokości ekranu
  3. 1 vmin jest równy 1% najmniejszej mierze ekranu
  4. 1 vmax jest równy 1% najmniejszej mierze ekranu

4. Dokładna analiza rozwiązań. Co łamie przypływ dokumentu

Jak można wyśrodkować horyzontalnie element HTML

  1. Użycie &nbsp; , aż element będzie na środku
  2. Ustawienie "width" i "margin-left" tak, aby element wydawał sie na środku
  3. Użycie "text-align: center"

Jak można wyśrodkować horyzontalnie element HTML

  1. Użycie "margin: 0 auto"
  2. Kombinacja "float" i "clearfix:both"
  3. Użycie kombinacji : position, left i transform

Czego nie chcemy. Analiza

  1. &nbsp; i "width" z "margin-left. Odpada
  2. text-align: center działa tylko dla "display:inline"
  3. margin: 0 auto działa tylko dla "display:block"

Czego nie chcemy. Analiza

  1. Najgorszy jest "float"
